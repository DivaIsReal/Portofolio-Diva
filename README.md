# Portofolio Diva Ahmad Pradana

Website portofolio pribadi milik **Diva Ahmad Pradana** — dibangun menggunakan Next.js, TypeScript, Tailwind CSS, dan Sanity CMS.

> 🔁 Project ini merupakan hasil rebrand dari portofolio milik [satriabahari](https://github.com/satriabahari/satriabahari.my.id). Terima kasih atas project open-source-nya yang menjadi fondasi website ini.

---

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Sanity CMS** — untuk manajemen konten projects
- **next-intl** — internasionalisasi (ID / EN)
- **Zustand** — state management
- **SWR** — data fetching

---

## Fitur

- Beranda dengan perkenalan & tombol Download CV
- Halaman About dengan riwayat karir & pendidikan
- Halaman Projects terintegrasi dengan Sanity CMS
- Dashboard (CTF History, HackTheBox, TryHackMe, PicoCTF)
- Halaman Achievements
- Network Monitor widget dekoratif
- Dark mode & multi-bahasa

---

## Menjalankan Lokal

```bash
# Install dependencies
bun install

# Jalankan dev server
bun run dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser.
